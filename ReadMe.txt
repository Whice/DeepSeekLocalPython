1. Установка Ollama (это надо самостоятельно). Официальный сайт: https://ollama.com/
2. Можно после установки проверить версию введя в консоли: ollama --version
3. Загрузка нужной версии для deepseek: ollama pull deepseek-r1:1.5b
Системные требования и версии можно посмотреть в файле DeepseekVersionRequrements.jpeg
Например варианты доступные сейчас для домашних ПК:
 ollama pull deepseek-r1:1.5b - Видеокарта с 0.75GB видеопамяти
 ollama pull deepseek-r1:7b - Видеокарта с 3.5GB видеопамяти
 ollama pull deepseek-r1:8b - Видеокарта с 4GB видеопамяти
 ollama pull deepseek-r1:14b - Видеокарта с 7GB видеопамяти
 ollama pull deepseek-r1:32b - Видеокарта с 16GB видеопамяти

Загрузка займёт определенное время, бОльшая модель будет грузиться дольше, т.к. весит больше. Тестировать лучше на 1.5b

4. Для python установить/обновить pip до последней версии.
5. Загрузить Ollama библиотеку: pip install ollama
6. Можно запускать скрипты python. (Может потребоваться доставить библиотеки для других функций, asyncio, к примеру)
